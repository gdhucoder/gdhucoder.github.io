2019-01-23随笔

“坚持” “努力” 今年希望把 “坚持” “努力” 这两个词从我的字典中删掉。 通过坚持和努力才能做的事情是做不好的。因为需要坚持的事情，我们从心里是不愿意做的。 不愿意做的事情怎么能够做的好呢？ 解决这个问题的技巧有三个： 1、 赋予这件事崇高的意义 2、如果不做这件事，我会失去什么 3、加入一个社群

How to rename remote git git remote set-url origin new_url ```shell $ git remote -v # View existing remotes > origin https://github.com/OWNER/REPOSITORY.git (fetch) > origin https://github.com/OWNER/REPOSITORY.git (push) $ git remote rename origin destination # Change remote name from 'origin' to 'destination' $ git remote -v # Verify remote's new name > destination https://github.com/OWNER/REPOSITORY.git (fetch) > destination https://github.com/OWNER/REPOSITORY.git (push) Written with StackEdit.
